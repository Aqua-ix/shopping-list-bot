# 買い物リスト bot

買い物リストを作成して確認できるLINE botです。以下のQRから友達追加ができます。

![img](asset/QR.png)

## 使い方

### リストの追加

買う予定のものを入力していきます。

<img src="./asset/add.png" width="200px">

### リストの確認

「リスト確認」で今までのリストを確認できます。

<img src="./asset/confirm.png" width="200px">

### リストのクリア

「リストをクリア」でリストの中身をリセットできます。

<img src="./asset/clear.png" width="200px">
